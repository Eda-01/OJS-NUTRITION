# 🥗 OJS Nutrition - Frontend

OJS Nutrition, kullanıcıların beslenme ve ürün takibini yapabileceği modern bir web arayüzüdür.  
Bu proje, frontend mimarisi ve component yapısını öğrenmek amacıyla geliştirilmiştir.

---

## 🚀 Kullanılan Teknolojiler

- React
- TypeScript
- Vite
- Axios (API istekleri için)
- CSS / SCSS

---

## 📂 Proje Yapısı

src/
│
├── components/       # Tekrar kullanılabilir UI bileşenleri
├── pages/            # Sayfa bileşenleri
├── services/         # API istekleri
├── hooks/            # Custom hook yapıları
└── utils/            # Yardımcı fonksiyonlar

---

Projeyi lokal ortamda çalıştırmak için:
